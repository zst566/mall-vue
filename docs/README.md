## 🎯 快速开始

### 新页面开发流程

1. **阅读设计规范**
   - 先阅读 [UI设计规范](./UI_DESIGN_SYSTEM.md)
   - 了解颜色、字体、间距等基础规范

2. **参考页面模板**
   - 查看 [首页布局设计](./HOME_PAGE_LAYOUT.md)
   - 参考现有页面实现

3. **使用封装组件**
   - 参考 [组件封装指南](./COMPONENT_GUIDE.md)
   - 使用统一的组件确保风格一致

4. **遵循代码规范**
   - 使用样式变量而非硬编码
   - 保持TypeScript类型完整
   - 遵循组件封装原则

## 📋 设计系统概览

### 品牌定位

- **品牌主色**: 蓝色系（`#1989fa`）
- **目标用户**: 青年及中年用户群体（25-50岁）
- **设计风格**: 现代简约、年轻活力

### 核心规范

- **颜色系统**: 蓝色主色 + 功能色 + 中性色
- **字体系统**: 系统字体，14px基准
- **间距系统**: 4px基准，12px/24px常用
- **圆角系统**: 8px默认，12px卡片
- **阴影系统**: 5级阴影层级
